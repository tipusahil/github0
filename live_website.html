kuno project/website er live website jevabe open/make korte hobe.

1.kuno project live website dekte caile protome jei project ta live korte cai seta github e giye save korte hobe, tar jonno 3ta step follow korte hobe.
  • 1.oi project open kore settings e giye (pages) e jaite hobe.
  • 2.then pages e jawar pore (branch) e giye (main)branch select kore save button e click korte hobe,
  • 3.erpore save hole github theke ekta live website er link banai dibe, jeta click korle oi project/(jetar live website dekte cacci) setar link dibe,
  • project/website er jodi github theke live er jonno link na dei thle nije thekew banai newa jai.

jodi github theke live website er link na dei thle j j ste p follow kore nije nije live website er link banate hobe ta holo:

  • 1.jei project tar live website dekte cai sei project ta github e open korte hobe, ortat github e push korar pore sei project ta open korte hobe, then open kore (ctrl+l) diye link ta copy kore nite hobe, (https://github.com/tipusahil/github0)
  • 2.then copy kora link ta arekta new tab niye okhane paste kore kisu edit korte hobe link e jmn: (https://github.com/tipusahil/github0) (https://tipusahil.github.io/github0) copy kora line theke 1.(username) ta github er age boshate hobe,2. & erpore/username er pore dot (.) dite hobe, 3.then & (com) er jaigai (io) likte hobe. 4.(.com)er pore j 2 ta (//) thake sekan theke ekta kete diye hobe. then enter press korlei live website show korbe. link er modde change korlew space dewa jbena.
